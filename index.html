GSAP - GreenSock Animation Platform

Apresentação Interativa sobre o framework GSAP

Desenvolvido para a disciplina de Web Design

O que é o GSAP?

GSAP é um framework JavaScript para criar animações de alta performance. Com ele, você pode animar qualquer coisa na web com facilidade.

Quando surgiu?

Criado em 2008 pela empresa GreenSock, o GSAP é hoje referência no mercado de animações web.

Para que serve?

Ideal para animar textos, imagens, SVGs, rolagens e interações. Muito usado em landing pages, apresentações e dashboards.

Estatísticas

Mais de 12 milhões de sites usam GSAP, incluindo grandes marcas como Google, Netflix e Microsoft.
